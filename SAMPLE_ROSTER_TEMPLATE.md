## How to Create Excel File:

1. Open Microsoft Excel or Google Sheets
2. Copy the table above
3. Paste into the spreadsheet
4. Save as `.xlsx` or export as `.csv`
5. Upload to the Duty Schedule system

## Tips:

- Keep column headers in the first row
- Don't leave the first row empty
- Make sure employee names are unique (or use ID column)
- Use consistent date format (YYYY-MM-DD recommended)
- Keep shift descriptions clear

Your file is ready to upload!
